# Xsens 动捕服 → GR-2 上半身遥操作使用说明

## 依赖

- Python 3.10+
- `fourier_aurora_client` (pip install fourier_aurora_client)
- `xsens_mvn_ros_python`（本仓库自带）
- MVN Studio（Windows 端，配置 UDP 输出）

## MVN Studio 配置

1. 打开 MVN Studio → Options → Network Streamer
2. 启用 UDP 输出，目标 IP 填运行 bridge 的机器 IP
3. 端口设为 `9763`（与 bridge 默认端口一致）
4. 勾选 Joint Angles、Quaternion 数据流

## 快速开始

### 1. MuJoCo 仿真模式（dry-run）

不连接 GR-2，只打印映射后的关节指令，用于验证映射是否合理：

```bash
cd /Users/puheliang/workspace/fmc3_xsens_mvn_ros
python3 -m gr2_xsens_bridge --dry-run --xsens-port 9763
```

程序会：
1. 等待 Xsens 数据包到达
2. 提示你站 T-pose 3 秒进行校准
3. 每 0.5 秒打印一次各关节指令值（弧度）

检查要点：
- 抬右臂 → `right_manipulator` 的 shoulder_pitch 应该变正
- 弯肘 → elbow_pitch 应该变负
- 转腰 → waist 值变化
- 如果方向反了，修改 `config.py` 中 `SIGN_MAP` 对应位置的 ±1

### 2. 连接 GR-2（实机或仿真）

```bash
# 默认参数
python3 -m gr2_xsens_bridge --xsens-port 9763 --domain-id 123

# 自定义 DDS domain ID
python3 -m gr2_xsens_bridge --xsens-port 9763 --domain-id 42
```

确保 `config.yaml` 中 `RunType` 设置正确：
- `0` = MuJoCo 仿真
- `1` = 实机

### 3. 运行方式

```bash
# 正常运行
python3 -m gr2_xsens_bridge

# 指定 Xsens 端口
python3 -m gr2_xsens_bridge --xsens-port 9765
```

## 命令行参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--xsens-port` | 9763 | Xsens UDP 端口 |
| `--domain-id` | 123 | GR-2 DDS domain ID |
| `--dry-run` | - | 不连接 GR-2，只打印指令 |

## 运行时行为

- 控制频率：50 Hz
- Xsens 数据超时 200ms → 冻结在最后安全位置，终端打印警告
- Ctrl+C → 2 秒内平滑回到零位，关闭连接，释放端口

## 调参指南

所有参数在 `gr2_xsens_bridge/config.py` 中：

- `SIGN_MAP`：关节方向符号，某个关节反了就改 1 → -1 或反过来
- `JOINT_LIMITS`：关节限位（弧度），防止超出机械极限
- `KP` / `KD`：电机 PD 增益，值越大跟随越紧但越容易震荡
- `SMOOTHING_ALPHA`：低通滤波系数，越小越平滑但延迟越大（当前 0.3）
- `MAX_JOINT_VEL`：最大关节角速度（rad/s），限制运动速度

## 常见问题

**端口被占用**
上次没正常退出导致的，等几秒重试即可（已设置 SO_REUSEADDR）。

**Xsens 数据收不到**
- 检查 MVN Studio 的 UDP 目标 IP 和端口
- 确认防火墙没有拦截 UDP
- 端口号必须 > 2000

**关节方向反了**
修改 `config.py` 中 `SIGN_MAP`，把对应位置的 1.0 改成 -1.0。

**动作太抖**
- 降低 `SMOOTHING_ALPHA`（比如 0.15）
- 降低 `KP` 增益
- 检查 Xsens 动捕服是否穿戴到位
